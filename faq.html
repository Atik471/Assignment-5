Important FAQ

  • DOM is an Application Programming Interface(API) that is used in a web browser for HTML and JavaScript to communicate with each other. DOM creates a tree like model of HTML docs where each HTML element is a node. So the JavaScript code can access each node(or element) with DOM methods and manipulate them.
  • We can select an element from DOM using the the DOM methods. The most used methods are:

    • document.getElementById() -> to select an element with it's id.
    • document.getElementsByClassName -> to select list of elements with it's class name.
    • document.getElementsByTagName -> to select list of elements with tag name
    • document.querySelector -> to select the first element with css selectors
    • document.querySelectorAll -> to select list of elements with css selectors
  • When an event occurs at a child element, it bubbles up the event to it's parent nodes. So the parent can add an action listener using the event. But the target of the event still is the child, which is the source. So instead of adding same event handler function to multiple child elements we can add a single event handler to the parent. This is called event delegation.

    Event delegation can be useful if child elements are generated dynamically. So that we don't need to explicitly add new event handler to each generated child.
  • Manipulating an element's attributes and styles can be done using DOM methods.

    To manipulate attributes, we can use these DOM method: getAttribute(attributeName), setAttribute(attributeName, value), removeAttribute(attributeName).

    To manipulate styles, we can manipulate classes or set inline styles. To manipulate classes we can use DOM methods like: classList.add(className), classList.remove(className), classList.toggle(className).
    To add inline styles this syntax can be used: element.style.property = 'value';
